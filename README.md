# 🐍 Snake Water Gun | Premium Web Edition

A modern, high-performance, and visually stunning implementation of the classic "Snake Water Gun" game (similar to Rock Paper Scissors). Reimagined with an **Apple-level UI**, glassmorphism, and smooth animations.

## ✨ Features

- **Premium UI/UX**: Designed with a focus on minimalism and elegance.
- **Glassmorphic Aesthetic**: Modern translucent cards with backdrop filters.
- **Responsive Design**: Play seamlessly on mobile, tablet, or desktop.
- **Python Backend**: Fast and efficient game logic powered by Flask.
- **Smooth Animations**: High-quality micro-animations for an interactive experience.

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or higher
- Flask (`pip install flask`)

### Installation & Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rajmund09/snake-water-gun-python.git
   cd snake-water-gun-python
   ```

2. **Run the application:**
   ```bash
   python main.py
   ```

3. **Open in your browser:**
   Navigate to `http://127.0.0.1:5000` to start playing!

## 🎮 How to Play

The rules are simple:
- **Snake** drinks **Water** (Snake wins)
- **Water** douses **Gun** (Water wins)
- **Gun** shoots **Snake** (Gun wins)

## 🛠️ Technology Stack

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Typography**: Inter (Google Fonts)
